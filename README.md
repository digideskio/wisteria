# Wisteria

![](img/logo.png)

A class-attribute based simple CSS framework.

## Install

```
bower install --save wisteria
```

```
npm install --save wisteria
```

## Browser Support

+ Google Chrome (latest)
+ Mozilla Firefox (latest)
+ Internet Explorer (latest)
+ Opera (latest)
+ Apple Safari (latest)
